# antifraud2gis-browser-extension
Browser (currently only chrome supported) extension for antifraud2gis. 

More on https://github.com/yaroslaff/antifraud2gis

Currently not available on chrome extensions market, and the project is still under development. 

## Installation (dev)

1. Go to chrome://extensions

2. Enable "Developer Mode"
![image](https://github.com/user-attachments/assets/045b7f5b-3034-4442-9f3b-ed79239fc2ed)

3. Click on "Load unpacked"
![image](https://github.com/user-attachments/assets/78baa683-8c7e-4539-8333-84248fa816cf)

4. Select all files from this repository (except for README)

5. Go to any 2gis website (depending on your location might be .ru/.kz/.kg etc)

6. Click on any place

7. A button should appear in top left corner to load reviews

8. In most cases the desired place/location might not be "loaded", so you will be prompted to visit external antifraud2gis page to load those reviews

9. Click load reviews, wait a bit, and get your result
